🎬 Movie Search App

A responsive and interactive movie search web application built using HTML, CSS, and JavaScript. This project uses the OMDB API to fetch real-time movie data and display detailed information in a clean user interface.

🚀 Features
🔍 Search movies by name
🎞️ Displays movie poster and title
📅 Shows release year
🎭 Displays genre and actors
⭐ IMDb rating display
📖 Movie plot description
📱 Responsive design for all devices


🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
OMDB API
